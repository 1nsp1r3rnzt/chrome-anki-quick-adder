Introduction

Welcome to Anki Quick Adder. This extension provides
  • Ability to create anki cards directly from Google Chrome on your Anki Desktop.
  • Dropdown with all your tag for autocomplete feature.
    • You can create card from
  • chrome Menu
  • Extension icon

    • Requirements

    You will require the following sofware to use the extension.

    1. Anki
    2. Anki Connect plugin
      3. Anki connect create a localhost server which is user to send requests to your Anki dekstop app.
    This extension is currently in beta. so, bear the bugs with me :-)
    Please, report bugs at github.

    Getting Started#back to top

    You can add cards through popup or chrome menu.

    Please, select a deck and note type before adding cards for the first time.

    From Chrome Menu

    Chrome Menu is available only, when Anki is running in background.

    If you cannot find this menu, then click here to read more.
    • Step 1 - Select the word, you want to enter
    • Step 2 - Click the field to send the word to that field.
    • Step 3 - Finally, click submit.
    • Step 4. Select, word for the other field and add it.
    • Step 6.Right click and Click submit.
    • Step 5. A popup will open to submit your data for 2 seconds and will close.

    Add From Popup.

    • Step 1 - Run Anki. The decklist is populated then only.
    • Step 2 - Click App icon to open app.
    • Step 3 - Select Deck type.
    • Step 4 - Select Card type.
    • Step 5 - Enter the data in fields.
    • Step 6(Optional) - Enter tags separated by ; for autocomplete.
    • Step 7 - Click Add Note.

    Inline editor

    Inline editor, is used to change color of text on the go.

    • Step 1 - Click app icon.
    • Step 2 - Write text in any field
    • Step 3 - Select the word to change color
    • Step 4. A toolbar will open to edit the word.

    Append Mode

    If you want to copy data, then append another data to it using chrome menu.

    • Step 1 - Switch on the append settings.
    • Step 2 - Select the word, you want to enter.
    • Step 3 - Click the field to send the word to that field.
    • Step 4 - Select another data.
    • Step 5. Right click and click on the same add to field
    • step 5. Click the submit button through popup or chrome menu.

    Known Bugs and Solutions #back to top

    No Chrome Menu - #back to top

    Click App icon>settings>Rebuild menu like on the screenshot.

    Potential Reasons

    1. Anki is not running
    2. Chrome was opened before running Anki

    Note Error - #back to top

    The Note is probably duplicate. Note might have front field empty

    No notification on Mac - #back to top

    Please read docs at Github page for information
